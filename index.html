What is AIMU?

All India Mistri Union ( shortly called AIMU ) is a chatting group on SoloLearn and it was created by Sudeep Jaiswal on May 2019. It was created with less than 10 but now, It has more than 20 members. Members of this group enjoys coding in various programming or non-programming languages. We ( AIMU members ) always ready for any challenges in either web or non-web code. Win or loss we never give up!

What does Jai AIMU mean?

You may have seen comments with #JaiAIMU tag by some of AIMU members, but you may don't know the meaning. Well, It is just a phrase used to express support for AIMU team. AIMU members often use this phrase in thier comment on any code to express the support to the author of code and the group.

What does Bawa / Mistri / Mistrani mean?

According to Tanay Bhatt, Bawa means brother, Mistri/Mistrani refers to bro/sis.

Can I join AIMU?

If we trust you, you will be added into the group.
